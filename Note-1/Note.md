## Distinct --> 
It is used it remove duplicate and repeated values from the result tables
### Syntax :-
    SELECT DISTINCT column1
    FROM table_name;

### QUESTIONS :-

## ALISE -->
It is an alternative name given to a column name or expression.     

### QUESTION :-
#### WQSTD THE ANNUAL_SAL

    - SELECT SAL*12 ANNUAL_SAL FROM EMP;
    - SELECT SAL*12 AS ANNUAL_SAL FROM EMP;
    - SELECT SAL*12 "ANNUAL_SAL" FROM EMP;
    - SELECT SAL*12 AS "ANNUAL_SAL" FROM EMP;


# SELECTION :-
  `WHERE`clause is use it to filter the records.
### character
  - `WHERE` clause execute after `FROM` clause.
  - `WHERE` clause execute row by row.
  - We can write multiple filter condition in `WHERE` clause by using logical operators.  

### Order of execution -->
  1. FROM
  2. WHERE
  3. SELECT

### QUESTIONS :-

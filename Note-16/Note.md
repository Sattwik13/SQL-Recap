#### P1. WAQTD the emp name and manager name and manager's manager name   

    SELECT E1.NAME,E2.NAME,E3.NAME
    FROM EMP E1, EMP E2, EMP E3

#### P2. WAQTD the emp ENAME, MGR NAME, EMP DNAME, MGR DNAME

#### P3. WAQTD THE EMP NAME, MGR NAME, MGR MGR NAME, EMP DNAME, MGR LOC, MGR MGR LOC.

#### 2.WAQTD ENAME, MANAGER'S SALARY AND MANAGER'S LOCATION IF MANAGER WORKS AS "PRESIDENT".

    SELECT E1.ENAME, E2.SAL, D.LOC
    FROM EMP E1, EMP E2, DEPT D
    WHERE E1.MGR=E2.EMPNO
    AND E2.JOB='PRESIDENT'
    AND E2.DEPTNO=D.DEPTNO;

#### 3. WAQTD ENAME, EMPLOYEE'S SALARY, MANAGER'S NAME,MANAGER'S SALARY ALONG WITH EMPLOYEES DNAME IF EMPLOYEE EARNS MORE THAN MANAGER. 

    SELECT E1.ENAME, E1.SAL, E2.ENAME, E2.SAL, D.DNAME
    FROM EMP E1, EMP E2, DEPT D
    WHERE E1.MGR=E2.EMPNO
    AND E1.SAL>E2.SAL
    AND E1.DEPTNO=D.DEPTNO;

#### 5.WAQTD ENAME, EMPLOYEE'S DNAME, MANAGER'S NAME, MANAGER'S DNAME IF EMP AND MANAGER WORKS INSAME LOC.

#### 6.WAQTD ENAME, EMPLOYEES' HIREDATE, MANAGER'S HIREDATE IF MANAGER WAS HIRED BEFORE THE EMPLOYEE IN TO ACCOUNTING DEPARTMENT.

    SELECT E1.ENAME, E1.HIREDATE, E2.HIREDATE

# OUTER JOJN -->

## LEFT OUTER JOIN :-
- It is used it to fetch the matching as well as unmatching record from the left table only.

    ### Syntax :-

    - <I>IMPLICIT</I>- 
        ```
        SELECT */COL_NAME/EXP
        FROM TAB_NAME1,TAB_NAME2
        WHERE TAB_NAME1.COL_NAME=TAB_NAME2.COL_NAME(+);

        SELECT * 
        FROM EMP,DEPT
        WHERE EMP.DEPTNO=DEPT.DEPTNO(+);
        ```
    - EXPLICIT-
        ```
        SELECT */COL_NAME/EXP
        FROM TAB_NAME1 LEFT JOIN TAB_NAME2
        ON TAB_NAME1.COL_NAME=TAB_NAME2.COL_NAME;
        ```

## RIGHT OUTER JOIN :-
- It is used it to fetch the matching as well as unmatching record from the right table only.

    ### Syntax :-

    - <I>IMPLICIT</I>- 

        ```
        SELECT */COL_NAME/EXP
        FROM TAB_NAME1,TAB_NAME2
        WHERE TAB_NAME1.COL_NAME(+)=TAB_NAME2.COL_NAME;

        SELECT * 
        FROM EMP,DEPT
        WHERE EMP.DEPTNO(+)=DEPT.DEPTNO;
        ```
    - <I>EXPLICIT</I>-

        ```
        SELECT */COL_NAME/EXP
        FROM TAB_NAME1 RIGHT JOIN TAB_NAME2
        ON TAB_NAME1.COL_NAME=TAB_NAME2.COL_NAME;
        ```

## FULL OUTER JOIN :-
- It is used it to fetch the matching as well as unmatching record from the BOTH table.

    ### Syntax :-

    - <I>EXPLICIT</I>-

        ```
        SELECT */COL_NAME/EXP
        FROM TAB_NAME1 FULL JOIN TAB_NAME2
        ON TAB_NAME1.COL_NAME=TAB_NAME2.COL_NAME;
        ```
